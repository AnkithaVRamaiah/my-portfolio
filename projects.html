2. Deploying Application on AWS EKS using Fargate and ALB

This project involved building a cloud-native application deployment on AWS using EKS and Fargate, eliminating the need to manage EC2 instances.

  • Ran Kubernetes pods on Fargate for serverless compute
  • Exposed the application over the internet using ALB with Ingress
  • Used OIDC and IAM roles for service accounts to securely allow only specific pods to access AWS resources

3. Load Balancing Python App with AWS ALB

Deploying a web application on a single instance creates a single point of failure and cannot scale efficiently. So I built a scalable solution to handle increased traffic and ensure high availability.

  • Deployed a Python-based web application on two private EC2 instances
  • Configured an Application Load Balancer to distribute traffic evenly between the instances
  • Improved fault tolerance, scalability, and performance of the app
